# Video Guide (Submission)

How to record the official Kaggle demonstration video for Oz AI.

## Requirements

| Item | Target |
|------|--------|
| Duration | 4:30 target, 5:00 maximum |
| Resolution | 1080p (1920×1080) |
| Audio | Clear narration, no background noise |
| Format | MP4/H.264 (platform-dependent) |

## Preparation

1. Complete [`../demo/recording_checklist.md`](../demo/recording_checklist.md)
2. Run `reset_demo.py` and verify all tabs populated
3. Rehearse using [`../demo/demo_flow.md`](../demo/demo_flow.md)

## Script structure

| Part | Duration | Content |
|------|----------|---------|
| 1 | 0:20 | Intro — problem, Oz AI, capstone track |
| 2 | 0:40 | Architecture — layers, ADK, MCP, Gemini, rules |
| 3 | 2:00 | Live demo — PowerShell incident, all tabs |
| 4 | 0:40 | Fallback — offline mode without API key |
| 5 | 0:40 | Closing — innovation, security, competition fit |

Full script: [`../demo/demo_script.md`](../demo/demo_script.md)  
Storyboard: [`../demo/storyboard.md`](../demo/storyboard.md)  
Narration notes: [`../demo/narration.md`](../demo/narration.md)

## Recording tips

- Browser zoom 100–110%, dark theme, hide bookmarks bar
- Disable notifications (Do Not Disturb)
- Never show API keys on screen
- Smooth cursor movement between tabs
- Record fallback segment separately if needed

## After recording

1. Verify duration ≤ 5:00
2. Upload to YouTube or Kaggle-hosted video
3. Add URL to submission notebook
4. Optional: captions from `narration.md`
